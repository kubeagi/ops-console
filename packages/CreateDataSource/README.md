## 开发

环境要求：

- **Node.js v18.x**
- **pnpm v8.x**

安装依赖：

```bash
$ npm i pnpm @antfu/ni -g
$ ni
```

运行项目：

```bash
$ nr dev
```

构建项目：

```bash
$ nr build
```

发布版本：

```bash
$ nr pub
```
